Flappypatrickio

View on GitHub Download .zip Download .tar.gz

HTML5 CSS-based Game

This game project has the following:

*1280x720 pixel game canvas. *All positions and sizes defined using a 10px em. *A simple game loop which calculates delta and can be started and stopped. *A player entity which can jump up when mouse is clicked or space bar is pressed. *A "Game Over" screen when player touches the ground or the obstacles. There you can see the highscore and restart the game.
